## ⚠️ Tips Penting

- Ukuran video sebaiknya di bawah **50MB** biar loading cepat di HP
- Foto sebaiknya di bawah **1MB** per foto (resize dulu di [squoosh.app](https://squoosh.app))
- Test di HP dulu!
- Pastikan nama file **sama persis**
